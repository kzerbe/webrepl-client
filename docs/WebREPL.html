execRaw(code, interval) → {Promise}

Evaluate the code on raw repl line by line and resolve promise when it gets executed (prints "OK"). If `interval` is passed, wait that amount of time before evaluating next line (important for ESP32 WebREPL).
Parameters:
Name Type Description
code string String containing lines of code separated by `\n`.
interval number Interval in milliseconds between the lines of code.
Source:
  • webrepl.js, line 193
Returns:
Resolves when code is pasted and executed (got `OK` from repl) on raw repl mode.
Type
Promise
Example
let repl = new WebREPL({ autoConnect: true })
let code = `for i in range(0, 10):\n    print(i)`
repl.on('authenticated', function() {
    this.enterRawRepl()
        .then(() => this.execRaw(code, 30))
        .then(() => this.exitRawRepl(code))
})
